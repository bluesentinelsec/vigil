# basl embed

Embed files and directories into importable `.basl` modules. Binary data is stored as base64-encoded strings with optional gzip compression. The generated module uses only the standard library (`base64`, `compress`) and works with the normal `import` system.

## Usage

```sh
basl embed <file|dir...> [-o output.basl] [--compress] [--no-compress]
```

## Modes

### Single File

```sh
basl embed logo.png                     # → logo_png.basl
basl embed config.json -o cfg.basl      # custom output name
```

Generated module API:

| Symbol | Type | Description |
|--------|------|-------------|
| `name` | `pub string` | Original filename (`"logo.png"`) |
| `size` | `pub i32` | Original file size in bytes |
| `bytes()` | `pub fn() -> (string, err)` | Decode and return the raw data |

```c
import "logo_png";

fn main() -> i32 {
    string data, err e = logo_png.bytes();
    fmt.println(logo_png.name);            // "logo.png"
    fmt.println(string(logo_png.size));     // "2048"
    file.write_all("output.png", data);
    return 0;
}
```

### Multiple Files

```sh
basl embed logo.png icon.ico -o assets.basl
```

Generated module API:

| Symbol | Type | Description |
|--------|------|-------------|
| `count` | `pub i32` | Number of embedded files |
| `get(path)` | `pub fn(string) -> (string, err)` | Get file data by name |
| `list()` | `pub fn() -> array<string>` | List all embedded file names |

```c
import "assets";

fn main() -> i32 {
    string data, err e = assets.get("logo.png");
    array<string> files = assets.list();
    fmt.println(string(assets.count));      // "2"
    return 0;
}
```

### Directory (Recursive)

```sh
basl embed ./assets/                    # → assets.basl
basl embed ./assets/ -o resources.basl  # custom output name
```

Walks the directory tree and embeds every file. Paths are relative to the directory root, always using forward slashes regardless of OS.

Same API as multiple files: `get(path)`, `list()`, `count`.

```c
import "assets";

fn main() -> i32 {
    // Paths are relative to the embedded directory root
    string json, err e = assets.get("config/app.json");
    string logo, err e2 = assets.get("images/logo.png");

    // List everything
    array<string> files = assets.list();
    for f in files {
        fmt.println(f);
    }
    // config/app.json
    // images/icon.ico
    // images/logo.png

    return 0;
}
```

## Compression

By default, `basl embed` auto-detects whether gzip compression is beneficial. If gzipping the file saves at least 10% of its size, the data is stored as gzip+base64. Otherwise, plain base64 is used.

| Flag | Behavior |
|------|----------|
| *(default)* | Auto-detect: gzip if it saves ≥10% |
| `--compress` | Always gzip before base64 |
| `--no-compress` | Always use plain base64 |

Compression is transparent to the caller — `bytes()` and `get()` handle decompression automatically.

### Size comparison for a 1 MB file

| Format | Approach | Output size |
|--------|----------|-------------|
| Already compressed (PNG, ZIP) | base64 only | ~1.33 MB |
| Compressible text (JSON, XML) | gzip + base64 | ~0.1–0.5 MB |
| `array<u8>` decimal (not used) | comma-separated bytes | ~3.5 MB |

## Naming

The output filename is derived from the input by sanitizing to a valid BASL identifier:

| Input | Output file | Import name |
|-------|-------------|-------------|
| `logo.png` | `logo_png.basl` | `logo_png` |
| `my-font.ttf` | `my_font_ttf.basl` | `my_font_ttf` |
| `data.tar.gz` | `data_tar.basl` | `data_tar` |
| `./assets/` | `assets.basl` | `assets` |

Use `-o` to override the output filename.

## Error Handling

`get()` returns an error for unknown paths:

```c
string data, err e = assets.get("nonexistent.txt");
// e == err("embed: not found: nonexistent.txt")
```

`bytes()` returns an error if decoding fails (should not happen with generated files):

```c
string data, err e = logo_png.bytes();
if (e != ok) {
    fmt.println(e.message());
}
```

## Generated File Format

Generated files are valid BASL source. They contain a `// do not edit` header, import the necessary stdlib modules, and define `pub` symbols for the embedded data. You can inspect, diff, and version control them like any other `.basl` file.

```c
// generated by basl embed — do not edit (gzip+base64)
import "base64";
import "compress";

pub string name = "config.json";
pub i32 size = 48210;

string _b64 = "H4sIAAAAAAAAA6tWKkktLlGyUl...";

pub fn bytes() -> (string, err) {
    string decoded, err e1 = base64.decode(_b64);
    string data, err e2 = compress.gunzip(decoded);
    return (data, e2);
}
```
