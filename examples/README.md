# VIGIL Examples

This directory contains practical examples demonstrating various features of the VIGIL language.

## Running Examples

```bash
vigil examples/hello.vigil
```

## Examples

### Basic Examples

- **hello.vigil** - Simple "Hello, World!" program
- **collections.vigil** - Working with arrays and maps
- **error_handling.vigil** - Error handling patterns with multi-return values
- **classes.vigil** - Object-oriented programming with classes

### I/O and Data

- **file_operations.vigil** - Reading, writing, and listing files
- **json_parsing.vigil** - Parsing and generating JSON
- **database.vigil** - SQLite database operations
- **regex_patterns.vigil** - Regular expression matching and manipulation

### Networking

- **http_server.vigil** - Simple HTTP server with routing

### Advanced

- **concurrency.vigil** - Multi-threading with mutexes
- **cli_tool.vigil** - Command-line argument parsing

## Learning Path

If you're new to VIGIL, we recommend following this order:

1. Start with `hello.vigil` to understand basic syntax
2. Explore `collections.vigil` for data structures
3. Learn error handling with `error_handling.vigil`
4. Try `file_operations.vigil` for I/O operations
5. Experiment with `classes.vigil` for OOP concepts
6. Build something practical with `http_server.vigil` or `database.vigil`
7. Explore advanced topics like `concurrency.vigil`
