# BASL Examples

This directory contains practical examples demonstrating various features of the BASL language.

## Running Examples

```bash
basl examples/hello.basl
```

## Examples

### Basic Examples

- **hello.basl** - Simple "Hello, World!" program
- **collections.basl** - Working with arrays and maps
- **error_handling.basl** - Error handling patterns with multi-return values
- **classes.basl** - Object-oriented programming with classes

### I/O and Data

- **file_operations.basl** - Reading, writing, and listing files
- **json_parsing.basl** - Parsing and generating JSON
- **database.basl** - SQLite database operations
- **regex_patterns.basl** - Regular expression matching and manipulation

### Networking

- **http_server.basl** - Simple HTTP server with routing

### Advanced

- **concurrency.basl** - Multi-threading with mutexes
- **cli_tool.basl** - Command-line argument parsing

## Learning Path

If you're new to BASL, we recommend following this order:

1. Start with `hello.basl` to understand basic syntax
2. Explore `collections.basl` for data structures
3. Learn error handling with `error_handling.basl`
4. Try `file_operations.basl` for I/O operations
5. Experiment with `classes.basl` for OOP concepts
6. Build something practical with `http_server.basl` or `database.basl`
7. Explore advanced topics like `concurrency.basl`
